## 📖 About the Project

AI Student Assistant is a modern web application designed to simplify students' academic work using the power of **Generative AI**.

Instead of switching between multiple websites for studying, note summarization, quizzes, emails, planning, and assignments, students can access everything from a single intelligent platform.

This project was developed as part of my **Summer Training Project (2026)**.

# 🚀 Installation

## Clone the repository

```bash
git clone https://github.com/HarshSingh-2310/AI-Student-Support-Service.git
```

---

## Move into the project

```bash
cd AI-Student-Support-Service
```

---

## Install dependencies

```bash
pip install -r requirements.txt
```

---

## Create a `.env` file

```env
GOOGLE_API_KEY=YOUR_GEMINI_API_KEY
```

---

## Run the application

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000
```
